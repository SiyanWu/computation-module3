About Wardrob - E

A mobile application that allows users to know and record their clothes styles so as to avoid waste of resources and textile pollution caused by repeated purchases of similar clothes.

The final deliverable will be a mobile application that can interact with users, and this web is designed for users to help them know the app more. Its main design will be as shown below, which are the home page and two main function display pages.

Function 1

Add new items to the existing categories:

Each plus icon represents a different type, such as hats, scarves, shoes and so on. After the user clicks the plus icon, new items can be intelligently scanned. Smart scan can only scan out part of the item information. If users want to complete the information, they still need to edit it by themselves. You can also write additional information in the notes column to facilitate keyword search function.

Function 2

Find items that users have deposited in the system:

After clicking the magnifying glass icon in the upper right corner of the main page, the plus icon of each category will become the category name. Click on the type you want to find, and the system will form a real electronic wardrobe scene for you. Click on the clothes you want to find to display the complete information and images of the item. Users can also use the search function in the upper right corner to accurately search based on information keywords.
